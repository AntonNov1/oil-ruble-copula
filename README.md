# Oil–Ruble Dependence: Copulas and Regime Switching

Bachelor's thesis, MIPT FPMI, 2026.

The short version: I looked at how the relationship between Brent oil prices and the ruble exchange rate changed over 2017–2025, and whether you can describe that with copulas instead of plain correlation. Turns out you can, and the sanctions shock of 2022 shows up very cleanly.

The compiled PDF is [`thesis.pdf`](thesis.pdf).

---

## What's in here

```
scripts/    R pipeline, one script per stage (00–15)
tex/        LaTeX source + figures + bibliography
thesis.pdf  final compiled version
run_all.R   runs scripts 01–13 in sequence
```

### Scripts

| Script | What it does |
|--------|-------------|
| `00_helpers.R` | shared functions used everywhere |
| `01_data_prep.R` | load Brent + USD/RUB, compute weekly log-returns |
| `02_structural_breaks.R` | find break points in the joint distribution |
| `03_copula_estimation.R` | fit Frank / Gumbel / Clayton / Gaussian / Student to each period |
| `04_gof_tests.R` | goodness-of-fit per period |
| `05–08` | cross-validation, robustness, rolling window, forecasting |
| `09_regime_copulas.R` | greedy set-cover: which 3 copulas cover all 6 periods? |
| `10_homogeneity_test.R` | LRT + bivariate KS: can periods with the same copula be pooled? |
| `11_gof_pooled.R` | CvM test on pooled data |
| `12_regime_switching.R` | 3-state HMM (Hamilton filter + Kim smoother) |
| `13_em_algorithm.R` | extend to 4 states via Baum–Welch EM |
| `14–15` | regime-based trading strategy (weekly and daily data) |

## The main findings

Five structural breaks split the sample into six periods. Three copulas are enough to cover all of them:

- **Frank** (θ = 2.51) for 2017–2022 — symmetric moderate positive dependence, the "normal" regime
- **Student-t** (ρ = −0.63, ν = 0.56) for the few weeks around the Feb 2022 sanctions — heavy tails, sharp negative dependence
- **Rotated Gumbel** (θ = 1.09) for 2022–2025 — weak negative dependence, basically the oil-ruble link is gone

The HMM on top of these three (script 12) gets 93% agreement with the historical break dates using only the data, no external labels. AIC = −30.

Testing a 4-state model with full EM re-estimation (script 13) doesn't help — the extra state doesn't survive the likelihood ratio test (p = 0.63), so three is the right number.

## Running it

You'll need R ≥ 4.2 and these packages:

```r
install.packages(c("copula", "dplyr", "tidyr", "ggplot2",
                   "quantmod", "httr", "xml2", "scales"))
```

Data is downloaded automatically on first run (Brent from Yahoo Finance, USD/RUB from the Bank of Russia XML API), so you need internet access.

Set your working directory to the project root before running anything, then:

```r
source("run_all.R")          # full pipeline
source("scripts/12_regime_switching.R")   # just the HMM part
```

Results go to `results/<script_number>/`.

To recompile the thesis:

```bash
cd tex && make
```

Requires MacTeX or TeX Live with biber.

## Data

Weekly and daily closing prices: Brent crude (EIA/Yahoo Finance) and USD/RUB spot (Bank of Russia), August 2017 – April 2026. Raw CSVs are not in the repo — they're regenerated by `01_data_prep.R`.
