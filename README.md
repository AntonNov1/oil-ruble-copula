## What this is about

The ruble–oil relationship is one of the most studied links in Russian macroeconomics. Before 2022, rising oil prices reliably strengthened the ruble; after the sanctions shock of February 2022, that link broke down. This work treats the **dependence structure itself** — not just the correlation — as the object of study.

The approach:
1. Detect structural breaks in the joint distribution of weekly Brent and USD/RUB log-returns
2. Fit a separate bivariate copula to each stable sub-period
3. Show that three copula families (Frank · Student-t · rotated Gumbel) cover all six periods
4. Build a hidden Markov model (HMM) over those regimes and estimate it by maximum likelihood, then extend to a full EM (Baum–Welch) estimation

The compiled thesis PDF is at [`thesis.pdf`](thesis.pdf).

## Key results

### Structural breaks and copula selection

Five breaks were found in the joint distribution (KS test, α = 1%):

| Break date | Event |
|------------|-------|
| 2020-01-18 | Oil slide before COVID crash |
| 2020-06-20 | OPEC+ production cut deal |
| 2022-02-26 | Sanctions shock |
| 2022-06-11 | Ruble stabilisation |
| 2023-09-09 | New oil export restrictions |

Three copulas cover all six periods by the greedy set-cover criterion (AIC < 0 on pooled data):

| Copula | Periods | τ | Interpretation |
|--------|---------|---|----------------|
| Frank (θ = 2.51) | P1–P3 | +0.25 | Symmetric moderate positive dependence |
| Student-t (ρ = −0.63, ν = 0.56) | P4 | −0.39 | Heavy-tailed negative, sanctions crisis |
| Gumbel 90° (θ = 1.09) | P5–P6 | −0.09 | Weak negative, decoupled market |

### HMM regime-switching (script 12)

Transition matrix estimated by Nelder-Mead on the full sample:

|  | → Frank | → Student | → Gumbel° |
|--|---------|-----------|----------|
| Frank | 0.949 | 0.048 | 0.003 |
| Student | 1.000 | 0.000 | 0.000 |
| Gumbel° | 0.003 | 0.000 | 0.997 |

- LogLik = 21.06 · AIC = −30.13
- Viterbi path agrees with deterministic break boundaries in **93%** of weeks

### EM extension (script 13)

The 4-state Baum–Welch model (adding a Gumbel upper-tail regime) does not improve fit:
- ΔlogLik = +4.5, but ΔAIC = +13 (17 parameters vs 6)
- LR test: λ = 8.94, df = 11, **p = 0.63**

Three states are sufficient.

## Reproducing the results

### Requirements

```r
install.packages(c(
  "copula", "dplyr", "tidyr", "ggplot2",
  "quantmod", "httr", "xml2", "scales"
))
```

R ≥ 4.2 recommended. All scripts tested on macOS with R 4.4.

### Running

```r
# run everything in order
source("run_all.R")

# or individual stages
source("scripts/09_regime_copulas.R")
source("scripts/12_regime_switching.R")
source("scripts/13_em_algorithm.R")
```

Output goes to `results/<script_name>/`.

### Compiling the thesis

```bash
cd tex
make        # full build: pdflatex + biber + pdflatex × 2
make fast   # single pdflatex pass (no bibliography update)
make view   # open PDF
```

Requires a TeX Live or MacTeX installation with `biber`.
